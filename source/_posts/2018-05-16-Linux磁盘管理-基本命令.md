## df
```
df：查看磁盘分区使用状况
-l：仅显示本地默认磁盘（默认）
-a：显示所有文件系统的磁盘使用情况
-h：以1024进制计算最合适的单位显示磁盘容量（默认KB显示）
-H：显示以1000进制计算最合适的单位显示磁盘容量
-T：显示磁盘分区类型
-t：显示指定类型文件系统的磁盘分区  //如：df -t ext4
-x：不显示指定类型文件系统的磁盘分区
```



## du
```
du 统计磁盘上的文件大小
-b 以byte为单位统计文件  //统计出来的都是四舍五入的大小
-k 以KB为单位统计文件    //统计出来的都是四舍五入的大小
-m 以MB为单位统计文件    //统计出来的都是四舍五入的大小
-h 按照1024进制以最适合的单位统计文件
-H 按照1000进制以最适合的单位统计文件
-s 指定统计目标  
du -s tmp
du -sb *.zip
```

## 定义变量 
```
变量名=变量值
#=号左右不可有空格
#变量值有空格需要加""
#所有变量默认的值都是字符串类型，不可做Math运算

x=5
x="hello jx"
```
<br>

## 调用变量 
```
$变量名

echo $x
```
<br>

## 变量叠加
```
x=123           #123
x="$x"456       #123456
x=${x}789       #123456789

x=5
y=6
z=$x+$y         #5+6
```


<br>

## 变量查看
```
set
选项：
-u：如果设定此选项，调用未声明变量里会报错（默认调用未声明变量无任何提示，值为空的变量也无任何提示）

echo $jflajlf
set -u
echo $jflajlf
```
<br>

## 删除变量
``` 
unset 变量名
#注意变量名没有$，因为要删除的是变量，而不是变量的值

unset x
```

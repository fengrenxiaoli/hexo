1. tags标签使用[tag1,tag2,...]，注意冒号后需要一个半角空格

1. 文章内标题使用`###空格我是标题空格`

3. 内嵌代码使用反引号
```
`我是代码`
```

<!--more-->

4. 其他代码使用```键

5. 文字加粗使用两个星号
```
**我是文字**
```
6. 文字斜体使用一个*星号
```
*我是文字*
```
7. 引用使用>
```
>我是文字
```
8. 链接使用以下方式
```
[显示文字](我是链接)
```
9. 表格内使用竖线用`&#124;`

## Github
1. 创建SSH Key
```
cd 
ssh-keygen -t rsa -C "youremail@example.com"
cd .ssh
```

2. 登陆GitHub，打开“Account settings”，“SSH Keys”页面，在Key文本框里粘贴`id_rsa.pub`文件的内容

GitHub需要识别出你推送的提交确实是你推送的，而不是别人冒充的

不要把敏感信息放进去
